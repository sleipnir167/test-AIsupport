# AIテスト支援システム - モックアプリ

Shift株式会社が開発するAI活用テスト支援Webアプリケーションのモックアップです。

## 概要

DX推進部門向けのE2Eテスト支援システム。顧客の要件定義書・機能設計書・Webサイト構造・ソースコードを読み込み、AIが品質保証プロフェッショナルとしてテスト項目書を自動生成します。

## 機能

- 📁 **ドキュメント管理** - 要件定義書・設計書・QAナレッジのアップロード・管理（RAG）
- 🌐 **URLサイト構造分析** - 対象WebアプリのURL解析・UI要素抽出
- 💻 **ソースコード取込** - ソースコード解析・RAGインジェスト
- 🤖 **AIテスト項目生成** - LLMによる網羅的なテスト項目の自動生成
- 📋 **テスト項目書表示・編集** - インライン編集・フィルタ・カテゴリツリー
- 📊 **Excel出力** - テスト項目書のExcelダウンロード

## 技術スタック

- **Frontend**: Next.js 14 (App Router) + TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開きます。

## ディレクトリ構成

```
src/
├── app/                    # Next.js App Router
│   ├── login/              # ログイン画面
│   ├── dashboard/          # ダッシュボード
│   └── projects/[id]/      # プロジェクト詳細・各機能画面
├── components/
│   ├── ui/                 # 汎用UIコンポーネント
│   ├── layout/             # レイアウトコンポーネント
│   └── features/           # 機能別コンポーネント
├── lib/                    # ユーティリティ・モックデータ
└── types/                  # TypeScript型定義
```

## 注意事項

本アプリはモックアプリです。実際のAPI連携・データ永続化は実装されていません。
モックデータを使用してUIの動作確認が可能です。
